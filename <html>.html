📌 Introduction

A computer is an electronic device used to process data and generate useful information. It consists of various hardware components that work together efficiently.

⌨️ Input Devices

Input devices are used to enter data into the computer.

  • Keyboard
  • Mouse
  • Scanner
  • Microphone

🖥️ Output Devices

Output devices display processed data to the user.

  1. Monitor
  2. Printer
  3. Speaker
  4. Projector

⚙️ CPU

CPU is the brain of the computer.

  • ALU
  • Control Unit
  • Registers

🧠 RAM

RAM is a type of primary memory used to store data temporarily while the computer is running. It is volatile, meaning data is lost when power is turned off.

Feature RAM ROM
Type Volatile Non-volatile
Speed Fast Slow

🎮 GPU

GPU is responsible for rendering images, videos, and animations. It is especially important for gaming, video editing, and graphic design.

🧩 Motherboard

The motherboard is the main circuit board that connects all components of the computer.

  • Holds CPU and RAM
  • Connects all hardware parts
  • Contains expansion slots
  • Controls communication

💾 Storage

Storage devices store data permanently or temporarily.

Type Example
Primary RAM
Secondary Hard Disk
Portable Pen Drive

🔌 PSU

PSU supplies power to all components of the computer by converting AC to DC.

  • Provides electrical power
  • Ensures stable operation
  • Available in different watt capacities

Input → Process → Output cycle
